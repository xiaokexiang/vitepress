# GitHub Card 测试

## 使用方法

使用标准 Markdown 语法，链接文本必须是 `GitHub`（不区分大小写）：

[GitHub](https://github.com/xiaokexiang/some-repo) [GitHub](https://github.com/vuejs/core) [GitHub](https://github.com/vitejs/vite)

## 验证清单

- [ ] Markdown 链接自动转换为卡片
- [ ] 显示 `owner/repo` 路径
- [ ] 点击卡片打开 GitHub
- [ ] 多个卡片紧凑并排显示
